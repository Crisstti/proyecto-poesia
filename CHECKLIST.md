## 📋 Para Ejecutar el Proyecto

```bash
# 1. Instalar dependencias
npm install

# 2. Configurar .env.local con credenciales de Appwrite
cp .env.example .env.local
# Editar con tus valores

# 3. Ejecutar en desarrollo
npm run dev

# 4. Build para producción
npm run build

# 5. Preview del build
npm run preview
```
